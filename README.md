# Data_Comparative_Assessment_of_NL_Analysis_Software_for_-PBD_of_RC_Buildings_Part_2
Dataset Title:
Comparative Assessment of Nonlinear Analysis Software for Performance-Based Design of Reinforced Concrete Buildings - Part 2: System-Level Dynamic Analysis

Authors:
Kristijan Kolozvari
Saman A. Abdullah
Jeff Dragovich
Andrés Lepage

Description

This dataset contains the input data used in the numerical analyses reported in the paper “Comparative Assessment of Nonlinear Analysis Software for Performance-Based Design of Reinforced Concrete Buildings - Part 2: System-Level Dynamic Analysis,” submitted to Earthquake Spectra (2026).

A three-dimensional nonlinear finite element models were developed in ETABS, PERFORM 3D and OpenSees to capture nonlinear behavior the superstructure, and to conduct systematic comparison between the three software. The dataset enables replication of the analyses results comparing the considered software, as well as verification of the presented engineering demand parameters (EDPs).

Folder Structure

1. ETABS/
Contains ETABS (v22.7.0) model of the considered reinforced concrete core wall building.

2. OpenSees-STKO/
Contains OpenSees (v.3.7.0) and STKO (v4.0) models of the considered reinforced concrete core wall building.

3. PERFORM3D/
Contains PERFORM3D (v10) models of the considered reinforced concrete core wall building. 

4. Ground_Motions_Data/
Provides the input ground motion acceleration time histories used in the analyses.
